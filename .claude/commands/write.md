Du hilfst einen Abschnitt der Bachelorarbeit zu schreiben oder zu überarbeiten.

**Kontext der Thesis:**
Lies CLAUDE.md für den aktuellen Projektzustand. Schreibe nur auf Basis tatsächlich vorhandener Daten
und dokumentierter Methodik.

**Abschnitte, bei denen du helfen kannst:**
- Einleitung / Motivation
- Verwandte Arbeiten (Related Work)
- Methodik (das Wichtigste — hier liegt das Betreuer-Feedback)
- Messaufbau / Vantage Point
- Ergebnisse (Deskriptiv + Interpretation)
- Diskussion / Limitierungen
- Fazit

**Wenn der Nutzer einen Abschnitt angibt:**
1. Frag: Deutsch oder Englisch?
2. Frag: Welche konkreten Daten / Ergebnisse sollen verwendet werden?
3. Schreibe einen Entwurf (2–4 Absätze)
4. Markiere mit [TODO: X] wo noch Daten fehlen oder Zahlen einzusetzen sind

**Stil:**
- Akademisch, präzise, keine Füllwörter
- Aktiv wo möglich ("Wir messen X" statt "X wird gemessen")
- Einheiten immer angeben (ms, %, n=)
- Methodische Einschränkungen ehrlich nennen (Single Vantage Point, n-Größe)
- Keine Behauptungen ohne Daten

Wenn kein Abschnitt angegeben ist: Frag welcher Abschnitt bearbeitet werden soll.
